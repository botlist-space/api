Bot

Bot

A class which represents a bot that is listed on the site.

Constructor

new Bot(bot)

Parameters:
Name Type Description
bot object Contains the bot information returned raw from the site.
Properties:
Name Type Attributes Description
id string The ID of the bot.
username string The username of the bot.
discriminator string The discriminator number of the bot.
shortDescription string A short description of the purpose of the bot.
fullDescription string A detailed description of the bot. Contains Markdown, may be blank.
approved boolean If the bot has been approved, this will be true.
certified boolean If the bot is certified, this will be true.
avatar string The Discord avatar hash provided from their API.
invite string The invite URL for the bot provided by the owner.
support string <optional>
 The support server code provided by the owner, could be null.
avatarChildFriendly boolean A boolean stating if the avatar should be considered child friendly.
library string The name of the library which powers the back-end of the bot.
prefix string A short prefix which is required to use the bot's commands.
owners Collection.<string, User> A collection of User classes which hold listed owner's information.
tags Array.<string> An array containing tags that the bot has assigned, may be empty.
vanity string <nullable>
 A string containing the vanity of the bot. May be null.
createdAt number A timestamp in milliseconds since the bot was added to the site.
updatedAt number A timestamp in milliseconds since the bot was last updated.
serverCount number <optional>
 The amount of servers that this bot is in, may be null.
shards Array.<number> <optional>
 The shards that the bot has, each value is a server count on that shard, may be null.
